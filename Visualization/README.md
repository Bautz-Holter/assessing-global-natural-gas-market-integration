# Visualization Code

Publication-ready plots and tables for the natural gas spot market analysis pipeline. Each script reads from `Model/model output/` and writes figures and tables to `Visualization/Plot outputs/`, organized by stage.

---

## Overview

The visualization pipeline mirrors the model pipeline:

- **Data Cleaning** — Missing data maps and sample construction audits
- **Descriptive Analysis** — Price distributions, correlations, spreads, extreme returns
- **Unit Root & Structural Break** — Break location plots, stationarity test tables
- **Kalman Filter (3 variants)** — State paths, integration tables, residual diagnostics

All plots are **publication-ready**: no embedded titles (use LaTeX captions), 300 DPI, LaTeX-compatible dimensions (7.2" wide), and serif fonts for consistency.

---

## Running Visualizations

Visualizations run automatically as part of each pipeline stage. To run manually:

```bash
Rscript "Visualization/Code/Data Cleaning.r"
Rscript "Visualization/Code/Descriptive Analysis.r"
Rscript "Visualization/Code/Unit Root and Structural Break.r"
Rscript "Visualization/Code/Kalman filter/Standard Kalman.r"
Rscript "Visualization/Code/Kalman filter/Standard Kalman Filter with oil.r"
Rscript "Visualization/Code/Kalman filter/Dummy Kalman Filter.r"
```

Or target individual Kalman variants via the Python CLI:

```bash
python run.py kalman --family m1   # standard (no oil)
python run.py kalman --family m2   # standard with oil
python run.py kalman --family m3   # dummy variable
```

Required packages: `dplyr`, `readr`, `tidyr`, `ggplot2`, `scales`, `patchwork`, `here`  
Optional: `gt` (table PNG export); `webshot2` and `chromote` are loaded by `gt` internally — install separately if table export fails.

## Key Concepts

### Beta (Price Transmission)
- **Beta ≈ 1:** Prices move together — consistent with market integration
- **Beta << 1:** Limited transmission — segmented or frictionous markets
- **Long-run beta = beta / (1 − phi1 − phi2):** Total effect after AR(2) dynamics

### Delta (Oil Linkage)
- **Delta ≈ 0:** Gas prices independent of oil
- **Delta > 0:** Oil-linked pricing — gas moves with crude beyond hub-to-hub transmission

### Dummy Kalman
- **Beta_eff = beta_base + sum[omega × step_dummy]**
- Compares sharp regime-shift integration against the gradual time-varying beta path
- Omega estimates show the magnitude and direction of integration change at each break date

## Design Format

All plots follow a consistent publication template:

- **No figure titles** — use LaTeX captions only
- **7.2" wide** (single column) or **7.2" × 8"** (multi-panel)
- **300 DPI PNG** with white background
- **Times New Roman fonts** for LaTeX compatibility
- **Minimalist table borders** (top, bottom, header rule only)
- **Consistent hub colors** across all figures: JKM dark, TTF blue, NBP green, HH orange, Brent rose

## Troubleshooting

**Table PNG export fails?**
`gt` tables are rendered via webshot2/chromote. If this fails, save manually:
```r
gt::gtsave(gt_obj, "output.png")
```

**Missing input files?**
Run the corresponding model script first. For example, run `Kalman Filter/Standard Kalman Filter.r` before `Kalman filter/Standard Kalman.r`.

**Adjust plot dimensions or colors?**
Edit `theme_latex_compact()` (or `theme_thesis()`) for dimensions and `hub_colors` / `state_colors` vectors for the color palette. Both are defined in the `# Visual identity` block near the top of each Kalman filter visualization script.
